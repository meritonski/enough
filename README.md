# Enough — Support / Privacy / Terms (GitHub Pages)

This repo hosts the official Apple-required pages for the Enough app:
- Support URL: /support.html
- Privacy Policy URL: /privacy.html
- Terms URL: /terms.html

## Setup
1) Create a GitHub repo and add these files.
2) GitHub → Settings → Pages → Deploy from branch (main / root).
3) Update the support email in:
- support.html
- privacy.html
- terms.html

## Apple Standard EULA
Terms page includes the Apple Standard EULA link:
https://www.apple.com/legal/internet-services/itunes/dev/stdeula/
